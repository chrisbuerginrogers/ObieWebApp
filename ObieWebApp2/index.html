 · Oberlin Acoustics Workshop

Tools for analyzing the vibrational characteristics of violins

A modular browser-based toolkit. Drop in a measurement file, get an interactive analysis. Everything runs locally in your browser — no uploads, no server.

📂 Data & File Tools

📈
TRF File Reader

Open one or more TRF or TSV measurement files and overlay their frequency responses on an interactive log/linear plot.

Tool · PyScript
🔀
Convolve It

Load an FRF CSV or TRF and convolve it with any WAV file. Full complex frequency-domain convolution with min-phase or zero-phase reconstruction. Play and save the result.

Tool · PyScript
