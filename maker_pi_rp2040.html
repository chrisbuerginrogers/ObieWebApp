🔌 Set Up Your Board

Plug the Maker Pi RP2040 into your computer with a USB-C cable. The board should power on — you may hear a startup beep and see LEDs flash briefly.
MicroPython already installed? Skip to step 3.
New board / blank flash? The board will appear as a USB drive called RPI-RP2. Download the MicroPython .uf2 firmware from micropython.org/download/rp2-pico and drag it onto that drive. The board reboots automatically.
Click Connect to Board below. Chrome will pop up a list of serial ports — choose the one labelled something like USB Serial Device or Raspberry Pi RP2040.
Hit ⚡ Flash the LEDs! and watch the 13 NeoPixel LEDs blink red. That's live MicroPython running on the chip.

What just happened?

Clicking Flash sends this MicroPython program to the board over USB and runs it immediately: 

import neopixel, machine, time

np = neopixel.NeoPixel(machine.Pin(18), 13)   # 13 NeoPixels wired to GP18

while True:
    for i in range(13):
        np[i] = (255, 0, 0)      # set LED i to red (R, G, B)
    np.write()                     # push to LEDs — always needed!
    time.sleep(0.4)

    for i in range(13):
        np[i] = (0, 0, 0)        # off
    np.write()
    time.sleep(0.4)

Easy

🌈 Rainbow Cycle

Make all 13 NeoPixels sweep smoothly through the rainbow, wrapping around continuously.

def hsv(h):   # h in 0..1 → (R,G,B)
    i = int(h*6); f = h*6-i
    p,q,t = 0, int(255*(1-f)), int(255*f)
    v = 60  # brightness
    return [(v,t,0),(q,v,0),(0,v,t),
            (0,q,v),(t,0,v),(v,0,q)][i%6]

j = 0
while True:
    for i in range(13):
        np[i] = hsv((i/13 + j/256) % 1)
    np.write(); j = (j+1) % 256; time.sleep(0.02)
Easy

🎵 Play a Melody

Use the buzzer to play a short tune — try "Twinkle Twinkle" or invent your own using note frequencies.

NOTES = {'C4':262,'D4':294,'E4':330,
         'G4':392,'A4':440,'C5':523}
SONG = ['C4','C4','G4','G4','A4','A4','G4']
BEAT = [  1,  1,  1,  1,  1,  1,  2]

bz = machine.PWM(machine.Pin(22))
for note, dur in zip(SONG, BEAT):
    bz.freq(NOTES[note]); bz.duty_u16(22000)
    time.sleep(0.22 * dur)
    bz.duty_u16(0); time.sleep(0.05)
bz.deinit()

⏱ Reaction Timer

Flash the LEDs after a random pause. Measure how fast the user presses Button 1. Green = fast, red = slow.

import random
time.sleep(random.uniform(2, 5))   # surprise!
np.fill((80,80,80)); np.write()   # GO signal
t0 = time.ticks_ms()

rt = time.ticks_diff(time.ticks_ms(), t0)
print(f"Reaction: {rt} ms")
c = (0,200,0) if rt < 350 else (200,0,0)
np.fill(c); np.write()
Medium

🎚 ADC Brightness Slider

Connect a potentiometer to GP26 and use its voltage to smoothly control the brightness of all LEDs.

adc = machine.ADC(machine.Pin(26))
while True:
    v = adc.read_u16()              # 0..65535
    b = int(v / 65535 * 180)      # scale to 0..180
    np.fill((0, b, b))             # cyan, variable bright
    np.write(); time.sleep(0.04)
Hard

🚦 Traffic Light State Machine

Implement a traffic light (Red → Green → Yellow → Red) with the LEDs. Button 1 acts as a pedestrian-crossing request that cuts the green phase short.

Use a state variable ('red', 'green', 'yellow'). In each state, set all LEDs to the right colour and wait out a timer using time.ticks_ms(). During the green state, also check btn.value()==0 and break early if the button is pressed. Transition to the next state in a loop.
Hard

🕐 Binary Clock

Display the current time in binary on the NeoPixels: 5 LEDs for hours, 6 for minutes, 1 blinking for seconds, 1 for AM/PM. Increment once per second with a timer interrupt.

Use time.localtime() for H, M, S. Extract bits with >> and & 1 — e.g. (hours >> bit) & 1. Map bits to LED indices 0–4 (hours), 5–10 (minutes), 11 (seconds blink), 12 (AM/PM). Use machine.Timer with period=1000 to tick every second without blocking.

Appendix A — Maker Pi RP2040 Board Overview

▼

Key Features

  • RP2040 dual-core ARM Cortex-M0+ @ up to 133 MHz
  • 2 MB on-board QSPI Flash, 264 kB SRAM
  • 13 × WS2812B (NeoPixel) RGB LEDs — GP18
  • 2 push-buttons — GP20, GP21
  • Piezo buzzer — GP22
  • 4-ch DC motor driver (2 × DRV8833)
  • 7 servo motor headers (3.3 V signal, external power)
  • 2 × Grove I2C — SDA/SCL on GP4/GP5
  • 1 × Grove UART — TX/RX on GP0/GP1
  • 3 × ADC — GP26, GP27, GP28 (0–3.3 V)
  • USB-C for power & programming
  • Separate motor power connector (up to 10 V)

Appendix B — Introduction to MicroPython

▼

What is MicroPython?

MicroPython is a lean Python 3 implementation that runs directly on microcontrollers — no operating system, no computer needed. It supports most of Python's core syntax (variables, loops, functions, lists, dicts, classes) plus hardware modules for GPIO, PWM, ADC, I2C, SPI, and UART.

You talk to it via a REPL (Read-Eval-Print Loop) over USB — just like a Python shell on your laptop, but running on the RP2040 at 133 MHz.

Essential syntax

# Variables & types
x = 42;  name = "RP2040";  flag = True

# Conditionals
if x > 10:
    print("big")
elif x > 5:
    print("medium")
else:
    print("small")

# for loop
for i in range(10):   # 0, 1, 2 … 9
    print(i)

# while loop — run forever
while True:
    pass

# Functions
def add(a, b):
    return a + b

# f-strings (print with values)
print(f"x = {x}, name = {name}")

Key hardware modules

import machine, time, neopixel

# ── Digital output ──
led = machine.Pin(25, machine.Pin.OUT)
led.value(1)       # HIGH (on)
led.value(0)       # LOW  (off)
led.toggle()        # flip state

# ── Digital input ──
btn = machine.Pin(20, machine.Pin.IN,
                  machine.Pin.PULL_UP)
v = btn.value()     # 0=pressed, 1=open

# ── PWM ──
pwm = machine.PWM(machine.Pin(22))
pwm.freq(1000)     # 1 kHz
pwm.duty_u16(32768) # 50% duty (0–65535)
pwm.duty_u16(0)     # silent / off

# ── ADC (analogue in) ──
adc = machine.ADC(machine.Pin(26))
raw = adc.read_u16()   # 0 .. 65535
v   = raw * 3.3 / 65535 # convert to volts

# ── Timing ──
time.sleep(1)          # 1 second
time.sleep_ms(250)     # 250 ms
t0 = time.ticks_ms()
elapsed = time.ticks_diff(time.ticks_ms(), t0)

NeoPixel LEDs — GP18 (×13)

import neopixel, machine

NUM = 13
np = neopixel.NeoPixel(machine.Pin(18), NUM)

np[0] = (255, 0,   0)    # LED 0 → red
np[5] = (0,   0, 255)    # LED 5 → blue
np.fill((0, 80, 0))       # all LEDs dim green
np.fill((0,  0, 0))       # all off
np.write()                  # ALWAYS call after changes or nothing updates

# Brightness tip: scale (R,G,B) values — (255,0,0) is full, (30,0,0) is dim

Push Buttons — GP20, GP21 (active-low)

import machine

btn1 = machine.Pin(20, machine.Pin.IN, machine.Pin.PULL_UP)
btn2 = machine.Pin(21, machine.Pin.IN, machine.Pin.PULL_UP)

# Polling (check in a loop)
if btn1.value() == 0:          # 0 = pressed, 1 = not pressed
    print("Button 1!")

# Interrupt-driven (non-blocking callback)
def on_press(pin):
    print("Pressed!")

btn1.irq(trigger=machine.Pin.IRQ_FALLING, handler=on_press)
# IRQ_FALLING = fires when pin drops from 1→0 (button pressed)
# IRQ_RISING  = fires when pin rises 0→1 (button released)

Piezo Buzzer — GP22

import machine, time

# ── Simple click (no pitch) ──
pin = machine.Pin(22, machine.Pin.OUT)
pin.value(1); time.sleep_ms(5); pin.value(0)

# ── Tonal buzz (PWM) ──
bz = machine.PWM(machine.Pin(22))
bz.freq(440)              # Hz  — A4 = 440, middle C = 262
bz.duty_u16(28000)        # ~43% duty; lower = quieter
time.sleep(0.5)
bz.duty_u16(0)             # silence — do NOT deinit mid-program
bz.deinit()                 # release PWM slice when fully done

# Note frequencies (Hz)
# C4=262  D4=294  E4=330  F4=349  G4=392  A4=440  B4=494  C5=523

Analogue Inputs — GP26 / GP27 / GP28

import machine

adc0 = machine.ADC(machine.Pin(26))   # channel A0
adc1 = machine.ADC(machine.Pin(27))   # channel A1
adc2 = machine.ADC(machine.Pin(28))   # channel A2

raw   = adc0.read_u16()              # integer 0 .. 65535
volts = raw * 3.3 / 65535           # convert to volts (3.3 V full-scale)
pct   = raw / 65535                  # fraction 0.0 .. 1.0

# On-chip temperature sensor (ADC channel 4)
ts  = machine.ADC(4)
raw = ts.read_u16()

DC Motor Driver (DRV8833)

import machine

# Motor 1: GP8 (IN1), GP9 (IN2)   Motor 2: GP10 (IN1), GP11 (IN2)
m1a = machine.PWM(machine.Pin(8));  m1b = machine.PWM(machine.Pin(9))
m1a.freq(1000);                     m1b.freq(1000)

# Forward at ~60% speed
m1a.duty_u16(39321); m1b.duty_u16(0)

# Reverse at ~60%
m1a.duty_u16(0);     m1b.duty_u16(39321)

# Coast (free-spin stop)
m1a.duty_u16(0);     m1b.duty_u16(0)

# Brake (hold)
m1a.duty_u16(65535); m1b.duty_u16(65535)

Servo Motors — GP12–GP18

import machine

servo = machine.PWM(machine.Pin(12))  # Servo on S1 header
servo.freq(50)                         # 50 Hz standard servo

def set_angle(pwm, deg):
    # 0° → ~1 ms pulse, 180° → ~2.4 ms pulse (tune max_duty for your servo)
    min_d, max_d = 1638, 7864          # out of 65535 at 50 Hz
    pwm.duty_u16(int(min_d + (max_d - min_d) * deg / 180))

set_angle(servo, 0)    # fully anti-clockwise
set_angle(servo, 90)   # centre
set_angle(servo, 180)  # fully clockwise

I2C — Grove connector (GP4 SDA / GP5 SCL)

import machine

i2c = machine.I2C(0, sda=machine.Pin(4), scl=machine.Pin(5), freq=400_000)

print([hex(a) for a in i2c.scan()])    # list devices on the bus

# Read 2 bytes from device at 0x48
data = i2c.readfrom(0x48, 2)

# Write 0xFF to register 0x01 of device 0x48
i2c.writeto_mem(0x48, 0x01, bytes([0xFF]))

UART — Grove connector (GP0 TX / GP1 RX)

import machine

uart = machine.UART(0, baudrate=9600, tx=machine.Pin(0), rx=machine.Pin(1))

uart.write(b'Hello\r\n')

if uart.any():
    data = uart.read()

Timers (periodic callbacks)

import machine, time

# Hardware timer — fires a callback every 500 ms
def tick(t):
    print("tick")

timer = machine.Timer()
timer.init(period=500, mode=machine.Timer.PERIODIC, callback=tick)
# timer.deinit()   # stop the timer

# One-shot timer (fires once after 2 s)
timer.init(period=2000, mode=machine.Timer.ONE_SHOT, callback=tick)
